# **App Name**: AppVerse Hub

## Core Features:

- App Listing & Display: Display apps in a card-based layout with key details (name, description, version, size, download links, tags).
- Admin Authentication: Secure admin login using Firebase Authentication (username: Ashu000, password: Ashu78) for accessing the admin panel.
- App Management (Admin): Admin panel to add new apps, edit app details, delete apps, upload APK/files, change app icons, toggle app visibility.
- Download & Social Links (Admin): Admin panel to update app download links, Telegram/Website buttons, and edit social media links in the footer.
- Real-time Updates: Implement real-time updates for app listings using Firestore to reflect changes without page reloads.
- Download Counter: Track and display download counts for each app, stored and updated in Firestore.
- AI-powered Description Assistant: An AI tool to help generate SEO-friendly short descriptions for the app listings, configurable via the admin panel.

## Style Guidelines:

- Primary color: Deep violet (#9400D3) to capture the mysterious, creative vibe of an 'universe'.
- Background color: Dark grayish-purple (#282233) for a modern, dark theme aesthetic with very low saturation.
- Accent color: Soft blue (#6A5ACD), slightly to the 'left' of violet in the color wheel, for highlighting key elements like download buttons.
- Body and headline font: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel. If longer text is present, pair with 'Inter' for the body.
- Use custom icons for app categories and social media links, with a consistent line style and subtle glow effect.
- Mobile-first responsive design with a card-based layout for app listings, ensuring optimal viewing on all devices.
- Smooth transitions and subtle animations on button hovers and page scrolling to enhance user experience.
